# Mobile Accessories Store

Django-проект для лабораторних робіт: інтернет-магазин аксесуарів для мобільних телефонів.

## Тема

Каталог товарів: чохли, навушники, зарядні пристрої, павербанки та захисне скло.

## Лабораторні

- Лаба 1: приклад ООП з батьківським і дочірнім класами у `shop/oop_demo.py`.
- Лаба 2: створено Django-проект.
- Лаба 3: створено аплікацію `shop`, шаблони та навігацію.
- Лаба 4: створено моделі, зв'язки між таблицями та адмін-панель.
- Лаба 5: головна сторінка з хедером, футером, меню та товарами з БД.
- Лаба 6: сторінки товару й категорії, фото товарів.
- Лаба 7: кошик, форма розсилки та оцінювання товарів.
- Лаба 8: реєстрація, вхід, вихід, кабінет, відновлення пароля кодом, DRY через `base.html`.

## Запуск

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
